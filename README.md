# bakeoff
Leibniz Algorithm for Pi on Multiple Language Platforms
<br>Inspired by https://github.com/niklas-heer/speed-comparison

### Recipe to Get Up and Running
* Install gcc.
* Install ldc2 version of D.
* Install zig.
* Install Go.
* Install Java version 21.
* Install Jacobin.

 ### Execution
 Bash script: run.sh
